# PressKardu Analytics Dashboard

## Overview
Data Analytics dashboard for PressKardu — a hyperlocal pressing/ironing service platform.

## Tools Used
- Python (Pandas, Numpy, Matplotlib)
- Power BI
- Dataset: 600+ simulated orders

## Key Insights
- Janakpuri highest revenue generating area
- Thursday busiest day for orders
- Saree & Suit highest revenue services
- Peak orders in December-January season

## Files
- `presskardu_orders.csv` — Order dataset
- `presskardu_shops.csv` — Shop dataset
- `Untitled2.ipynb` — Python analysis notebook
